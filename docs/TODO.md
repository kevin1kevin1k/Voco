## 持續性驗收原則

- [ ] 每次新增分類或板面時，都要同步確認首頁入口、資料檔、導覽與語音內容一致
- [ ] 每次新增視覺元件時，都要同步檢查點擊尺寸、對比、焦點與鍵盤操作
- [ ] 每次調整推薦規則時，都要更新測試案例與可解釋性說明
- [ ] 每次調整 PWA 設定或資產時，都要重新驗證 build、安裝與離線行為
